# Student Performance Predictor

A Flask-based machine learning web application that predicts student academic performance based on study habits, attendance, participation, and other educational factors.

## Features

- User Registration & Login
- Student Dashboard
- Machine Learning Score Prediction
- Personalized Performance Insights
- Prediction History Tracking
- Admin Dashboard
- Analytics & User Management
- Daily Prediction Limits

## Tech Stack

- Python
- Flask
- Scikit-learn
- HTML
- CSS
- JavaScript

## Project Highlights

The application collects academic and behavioral metrics such as:

- Study Hours
- Sleep Hours
- Attendance
- Previous Grades
- Class Participation
- Assignments Completed
- Revision Hours
- Extracurricular Activities

These inputs are processed by a machine learning model to generate performance predictions and recommendations.
